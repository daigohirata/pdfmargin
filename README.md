# pdfmargin
CLI tools to add margins in PDF files

## Installation
```
$ git clone https://github.com/daigohirata/pdfmargin ~/pdfmargin
$ cd ~/pdfmargin
$ pip install .
```

## Usage
```
# Display all available options
$ pdfmargin --help
```

## Uninstallation

To uninstall `pdfmargin`, run the following command:

```bash
pip uninstall pdfmargin
